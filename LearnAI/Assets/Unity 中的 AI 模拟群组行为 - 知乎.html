Unity 中的 AI 模拟群组行为

陶程
大盈若冲

AI（人工智能）的主旨本质上是让计算机能够像生物体一样，具有思考和决定的能力来执行某些特定操作。

人工智能在游戏中的作用是通过提供富有挑战性的竞争对象来让游戏更好玩，在游戏世界中行动逼真的有趣的非玩家角色（NPC）会让游戏更好玩。通过让这些NPC对游戏世界中不断变化的情形，产生对玩家来说足够合理、有意义的反应，来让它们看起来更加智能。

群组行为：

模拟鸟群行走或者人群行走的过程称之为群组行动

Demo:

给鸟加上一些行为：

public class Crow : MonoBehaviour {


	public Vector3 velocity = Vector3.forward;
	public float animRandomTime = 2f;
	private Animation anim;

	private IEnumerator Start () {
		anim = GetComponentInChildren<Animation> ();
		//每隔0到2秒之间的随机数等待
		yield return new WaitForSeconds (Random.Range (0, animRandomTime));
		anim.Play ();
	}
	
	void Update () {
		transform.Translate (velocity * Time.deltaTime, Space.World);
	}
}

这样，鸟会不同的动画朝着前面移动。

我们可以让鸟群朝一个方向运动：

public class Crow : MonoBehaviour {

	public Transform target;
	public float speed = 1;
	public float animRandomTime = 2f;
	private Animation anim;

	private IEnumerator Start () {
		target = GameObject.Find ("Target").transform;
		anim = GetComponentInChildren<Animation> ();
		//每隔0到2秒之间的随机数等待
		yield return new WaitForSeconds (Random.Range (0, animRandomTime));
		anim.Play ();
	}
	
	void Update () {
		transform.LookAt (target.position);
		transform.Translate (Vector3.forward * Time.deltaTime * speed);
	}
}

但是这样，鸟群会随着时间的推移越来越靠近，不符合大自然的真实场景。

鸟群飞行会有三个特征：

  • 分离
  • 队列
  • 聚集

这基本上都是动物的本能，每个生物都会与其他生物分离，有时也会聚集，同时我们也都是队列的一部分。



第一个图是判断在一定范围内，视为靠近，可施加一个力进行分离。第二个图是指如果发现个体飞行方向偏离了大部队的方向，就施加一个力进行调整。第三个图是个体发现自己离鸟群比较远的时候，选取就近的几个点的中心点，进行力的施加，实现聚集。之所以使用力而不是速度是因为如果是直接设置速度的话就无法模拟现实环境，因为万事万物都是有惯性的。

如果你想要让鸟朝向上图中的红色箭头，有两种方式。一种是直接在这个箭头的方向施力，另一种是用目标方向减去当前方向施力。分离和聚集使用了第一种，因为鸟群以同样的速度在飞行，可视为是相对静止。可认为当前方向上的速度为0，所以直接在目标方向施力。队列使用第二种是因为方向的偏移，并不能视为相对静止。

实现分离：

主要思路是每隔一段时间判断一次是否需要分离。通过Physics.OverlapSphere函数判断鸟周围固定数值半径存在的 Collider，从而获取到附近的鸟类。通过本身的坐标减去周围的鸟的坐标，然后加起来。最终根据牛顿第二定律，确定速度。

public class CrowAI : MonoBehaviour {

	public Vector3 sumForce = Vector3.zero;  //合力，下面三个力的综合作用

	public Vector3 separationForce = Vector3.zero;  //分离的力
	public Vector3 alignmentForce = Vector3.zero;  //队列的力
	public Vector3 cohesionForce = Vector3.zero;  //聚集的力

	public float separationDistance = 3;  //距离这个鸟有三米以内的鸟类，跟其他鸟进行分离
	public List<GameObject> separationNeighbors = new List<GameObject>();  //存储附近的鸟
	public float separationWeight = 1; //分离的力所占的比重
	public float m = 1; //鸟的质量
	public Vector3 velocity = Vector3.forward;  

	public float checkInteterval = 0.2f;

	public float animRandomTime = 2f;
	private Animation anim;

	void Start () {
		InvokeRepeating ("CalForce", 0, checkInteterval);
		anim = GetComponentInChildren<Animation> ();
		//每隔0到2秒之间的随机数等待
		Invoke ("PlayAnim", Random.Range (0, animRandomTime));
	}

	void PlayAnim(){
		anim.Play ();
	}

	void CalForce(){
		sumForce = Vector3.zero;
		separationForce = Vector3.zero;  //分离的力
		alignmentForce = Vector3.zero;  //队列的力
		  cohesionForce = Vector3.zero;  //聚集的力

		separationNeighbors.Clear ();
		Collider[] colliders =  Physics.OverlapSphere (transform.position, separationDistance);
		foreach (Collider c in colliders) {
			if (c != null && c.gameObject != this.gameObject) {
				separationNeighbors.Add (c.gameObject);
			}
		}
		//计算分离的力
		foreach(GameObject neighbor in separationNeighbors){
			//需要远离的方向
			Vector3 dir = transform.position - neighbor.transform.position;
			//距离越小，施加的力越大
			separationForce += dir.normalized / dir.magnitude;
		}
		if (separationNeighbors.Count > 0) {
			separationForce *= separationWeight;
			sumForce += separationForce;
		}

	}

	void Update(){
		//牛顿第二定律，计算加速度
		Vector3 a = sumForce / m;
		// v=at ,速度计算公式
		velocity += a * Time.deltaTime;
		transform.Translate (velocity * Time.deltaTime, Space.World);
	}
}

实现队列：

思路差不多，只是用向量的减法实现目标方向的偏移。

//计算队列的力
		colliders =  Physics.OverlapSphere (transform.position, alignmentDistance);
		foreach (Collider c in colliders) {
			if (c != null && c.gameObject != this.gameObject) {
				alignmentNeighbors.Add (c.gameObject);
			}
		}
		Vector3 avgDir = Vector3.zero;
		foreach (GameObject n in alignmentNeighbors) {
			avgDir += n.transform.forward;
		}
		if (alignmentNeighbors.Count > 0) {
			avgDir /= alignmentNeighbors.Count;
			alignmentForce = avgDir - transform.forward;
			alignmentForce *= alignmentWeight;   
			sumForce += alignmentForce;
		}

实现聚集：

聚集的判断范围和队列的一样。通过找到附近的鸟然后求出中心点。根据向量的减法求出当前鸟的目标方向。

		if (alignmentNeighbors.Count <= 0)
			return;
		Vector3 center = Vector3.zero;
		foreach (GameObject n in alignmentNeighbors) {
			center += n.transform.position;
		}
		center /= alignmentNeighbors.Count;
		Vector3 dirToCenter = center - transform.position;
		cohesionForce += dirToCenter;
		cohesionForce *= cohesionWeight; 
		sumForce += alignmentForce;

这样的话后面的鸟的速度会显得很慢，需要添加一个恒定的力来保证每只鸟的速度。

//保持恒定飞行的力
		Vector3 engineForce = startVelocity - velocity;
		sumForce += engineForce * 0.3f;

在正常情况下，鸟群都会有一个首领带着飞。那么可以设计一个点，让鸟群进行跟随。

Vector3 targetDir = target.position - transform.position;
sumForce += (targetDir.normalized - transform.forward) * speed;

然后对鸟群的行为进行一个优化，主要是对各个数值的调整。
